# black-box-breakout

Das ist das öffentliche GitHub Repository für das Black Box Breakout Tutorium zum Thema 'Lokale KI für kreative Projekte'.

Hier sind alle Dateien abglegt, mit denen wir im Tutorium arbeiten. Die Dateien sind in die einzelnen Unterrichtseinheiten gegliedert und können so auch asynchron nachgearbeitet werden. 

# Kursbeschreibung:
Wie können wir Künstliche Intelligenz praktisch in eigene Projekte integrieren? Von der Idee bis zum funktionsfähigen Prototypen: Im Fokus stehen lokale KI-Modelle, schnelle Entwicklungsprozesse (Rapid Prototyping) und Vibe Coding – also das intuitive Gestalten von Anwendungen mithilfe intelligenter Systeme.

Ziel ist es, dass Teilnehmende die grundlegende Funktionsweise von KI verstehen, verschiedene Modelle und Anbieter kennen lernen und so fundierte Entscheidungen für ihre eigenen kreativen Vorhaben treffen können. Zudem entdecken sie die Vorteile lokaler Modelle als datensichere und anpassbare Alternative zu Cloud-Diensten.

Das Tutorium ist eine Maßnahme für die Studierenden des Fachbereich 5, um sich lokale KI Modelle mit hoher Datenautonomie anzueignen und für Projekte nutzbar zu machen. Ziel ist nicht eine grundsätzliche Lehre im Bereich Prompten o.ä. für kommerzielle Modelle wie ChatGPT oder NanoBanana.
