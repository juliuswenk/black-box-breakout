# Slide Decks starten

In diesem Ordner liegen mehrere Slidev-Präsentationen. Öffne zuerst den Ordner im Terminal:

```bash
cd slides
```

## Intro-Deck starten

```bash
npx slidev slides.md
```

## Vibe-Coding-Deck starten

```bash
npx slidev vibe-coding-101.md
```

Danach öffnet Slidev die Präsentation im Browser. Falls der Browser nicht automatisch aufgeht, nutze die Adresse, die im Terminal angezeigt wird, meistens:

```text
http://localhost:3030/
```

Slidev beendest du im Terminal mit `Ctrl + C`.
